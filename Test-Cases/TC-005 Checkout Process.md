# TC-005 Complete Checkout Process

## Module

Checkout

## Objective

Verify that the checkout process is completed successfully with valid information.

## Preconditions

- User is logged in.
- At least one product is in the cart.

## Test Data

First Name: John

Last Name: Smith

Postal Code: 12345

## Test Steps

1. Add a product to the cart.
2. Open the cart.
3. Click "Checkout".
4. Enter First Name.
5. Enter Last Name.
6. Enter Postal Code.
7. Click Continue.
8. Click Finish.

## Expected Result

The order is completed successfully and the confirmation message is displayed.

## Status

Pass
