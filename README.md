# 🛒 SauceDemo Testing

## Project Overview

This repository contains a manual testing project for the SauceDemo e-commerce application.

The objective of this project is to validate core functionalities such as login, product management, shopping cart operations and checkout process.

The project demonstrates QA skills through test case design, bug reporting and testing documentation.

---

## Application Under Test

SauceDemo

https://www.saucedemo.com/

---

## Scope

The following modules were tested:

- Login
- Products
- Shopping Cart
- Checkout
- Logout
